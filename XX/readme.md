迷路探索問題
============

内容
----
ある迷路が与えられた時、スタートからゴールへの最短経路を求めてください。

条件
----
* 入出力はテキスト形式で行ってください
* 一度に動けるのは上下左右だけで斜めに移動はできません
* 最短経路が複数あった場合は、その内のどれかが出力されれば構いません
* 入力データのエラーチェックは必要ありません
* 使用するプログラム言語や OS は自由です　※得意な言語が良いと思います

入力データ
----------
+ `input.txt`:
  迷路のデータ（S：スタート、G：ゴール、*：壁）
